## 2. Poetry 설치

1. 우선 poetry를 설치한다

```bash
poetry install

# 설치 제대로 됬는지 버전 확인
poetry --version
```

1. 의존성 설치

```bash
poetry install

poetry show
```

1. cmd + ,를 눌러 Interpreter에서 아래 사진과 같이 의존성이 설치되면 성공
